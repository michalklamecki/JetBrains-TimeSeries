# English
Implement a model that predicts whether an incident will occur within the next H time steps based on the previous W steps of one or more time-series metrics. Use a sliding-window formulation and train the model using any standard machine-learning framework.

The applicant may use any suitable public dataset or generate a synthetic time series with labeled incident intervals (e.g. anomalies or threshold breaches). The emphasis is on correct problem formulation, model selection, training, and evaluation rather than dataset complexity or model size.

The solution should include a clear description of the modeling choices, the evaluation setup (including alert thresholds and metrics), and an analysis of the results. During follow-up, the applicant should be able to explain the design decisions, discuss limitations, and outline how the approach could be adapted to a real alerting system.
Preferably, solutions should be provided as a link to a public GitHub repository.

# Polish
Zaimplementuj model, który przewiduje, czy incydent wystąpi w ciągu następnych H kroków czasowych na podstawie poprzednich W kroków jednej lub więcej metryk szeregów czasowych. Użyj formuły okna przesuwnego i wytrenuj model za pomocą dowolnego standardowego frameworka uczenia maszynowego.

Wnioskodawca może użyć dowolnego odpowiedniego publicznego zbioru danych lub wygenerować syntetyczny szereg czasowy z oznaczonymi przedziałami incydentów (np. anomalie lub przekroczenia progów). Nacisk kładziony jest na poprawne sformułowanie problemu, wybór modelu, trenowanie i ocenę, a nie na złożoność zbioru danych lub rozmiar modelu.

Rozwiązanie powinno zawierać jasny opis wybranych opcji modelowania, konfiguracji oceny (w tym progów i metryk alertów) oraz analizę wyników. Podczas dalszych działań wnioskodawca powinien być w stanie wyjaśnić decyzje projektowe, omówić ograniczenia i nakreślić, jak podejście można dostosować do rzeczywistego systemu alertów.
Najlepiej, aby rozwiązania były udostępniane w formie linku do publicznego repozytorium GitHub.
